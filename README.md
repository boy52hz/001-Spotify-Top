1. คะแนน IMDB_Scose ค่าสูงสุดของ Marvel และ DC

5. ค่ายไหนที่สร้างรายได้ที่ได้จากภาพยนต์ได้มากกว่ากัน
6. ภาพยนต์เรื่องไหนที่มีเวลาฉายต่ำกว่า 100 นาที ทั้งฝั่งของ Marvel และ DC
